# 텔레그램 실시간 트레이딩 게임 PRD (Product Requirements Document)

## 1. 서비스 개요
- **서비스 형태**: 텔레그램 기반 실시간 가상 트레이딩 경쟁 게임
- **목적**: 실제 시장 가격(바이낸스 등)의 흐름에 맞춰 가짜 돈(게임머니)으로 매매하며, 유저 간 수익률 랭킹을 겨루는 게임입니다. 청산의 긴장감, 랭킹을 통한 과시욕, 제한적 채팅 권한을 융합하여 강력한 락인(Lock-in)과 수익화를 창출합니다.
- **핵심 차별점**: 실제 돈을 투자하거나 잃지 않지만, 현물/선물(레버리지) 플레이가 가능하여 실전과 동일한 감각과 심리적 자극을 줍니다.

## 2. 핵심 비즈니스 모델 (BM 3축)
1. **구독료 (월 정기결제)**
   - 일반 요금: 월 $29.99
   - 제휴 거래소 혜택가: 월 $7.99 (평생 할인)
   - *전략*: 단순한 할인이 아닌, $29.99를 내는 손해를 피하기 위해 거래소 래퍼럴 가입을 유도하는 **손실회피형 가격 설계**입니다.
2. **게임머니 추가 결제 (인앱 결제)**
   - 청산 시 소유한 게임머니 전액 소실. 다시 플레이하려면 추가 결제 필요.
   - 1억 게임머니 당 3,900원 책정 (텔레그램 Stars 결제 연동).
3. **거래소 래퍼럴(Referral) 수익**
   - 30여 개 이상의 글로벌 거래소(Binance, Bybit, OKX 등)를 지원하여 유저의 거래소 전환 장벽을 낮추고 래퍼럴 가입을 극대화합니다.

## 3. 핵심 시스템 및 기능 명세

### 3.1. 게임 트레이딩 엔진
- **초기 자금**: 매일 모든 유저에게 게임머니 100억 지급.
- **시장 연동**: 실제 가상화폐 거래소의 실시간 시세를 반영.
- **매매 시스템**: 가짜 돈을 이용한 현물 및 선물(레버리지 적용) 매매 로직.
- **청산 시스템**: 레버리지 플레이 중 일정 손실 도달 시 강제 청산 및 자금 몰수.

### 3.2. 래퍼럴 인증 및 자동화 시스템
- **인증 방식 (텍스트 중심)**: 스크린샷 오류를 막기 위해 텍스트 기반(거래소명, UID 등)으로 인증받습니다.
- **인증 처리 레벨 (내부 검증 로직)**:
   - *레벨 1 (자동)*: 공식 데이터 확인이 명확해 즉시 자동 승인.
   - *레벨 2 (반자동)*: UID 및 텍스트 조합으로 빠른 검증 후 승인.
   - *레벨 3 (심사)*: 수동 검증이 필요한 경우 "심사 중" 상태 전환 후 승인.
- **인증 플로우**: 지원 거래소 선택 -> 인증 폼 제출 -> 시스템 검증 -> 승인 시 텔레그램 DM 및 평생 할인($7.99) 플랜 부여.

### 3.3. 랭킹 및 VIP 채팅 시스템
- **랭킹 시스템**: 당일 트레이딩 수익을 기준으로 실시간 랭킹 산정.
- **채팅 권한 부여 (과시욕 자극)**:
   - 기본적으로 게임 내 전체 채팅은 금지되어 있습니다.
   - 매일 밤 특정 시간(21:50 ~ 24:00 중 10분)에 당일 수익률 **1등 ~ 10등** 유저에게만 한시적으로 채팅 권한을 부여합니다.
   - 수익률 서열화 및 커뮤니티 권력/과시 요소를 통해 랭킹 경쟁을 극대화합니다.

## 4. 유저 플로우 (User Journey)
1. **진입**: 텔레그램 봇을 통해 서비스에 접속합니다.
2. **거래소 선택 및 인증**: 30여 개 지원 거래소 중 이용 중인 곳을 선택하고 래퍼럴 UID를 제출합니다.
3. **구독 결제**: 인증 심사 통과 후 평생 할인가($7.99)로 구독을 완료합니다. (미인증 시 $29.99 결제)
4. **플레이**: 매일 100억 게임머니를 지급받고 실시간 시장 상황에 맞춰 트레이딩을 진행합니다.
5. **청산 및 재구매**: 무리한 레버리지로 청산당할 경우 텔레그램 Stars로 게임머니를 추가 결제합니다.
6. **경쟁 및 보상**: 매일 랭킹이 집계되며, 상위 10위 안에 들면 특정 시간에 채팅방에서 과시할 수 있는 권한을 얻습니다.

## 5. 기술 스택 요구사항 (Target Stack)
- **Frontend**: React (Functional Components & Hooks), TypeScript, Tailwind CSS
- **Backend**: Node.js, TypeScript
- **Platform**: Telegram Web App 연동 및 Telegram Bot API
- **결제 연동**: Telegram Stars API 및 구독 결제 모듈 연동
